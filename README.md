# 🚀 Willian Kelvin | Engenharia de Software

Atualmente cursando o **1º semestre de Engenharia de Software**, com foco total na construção de uma base técnica sólida e no aprendizado das melhores práticas do mercado. Meu objetivo é evoluir como desenvolvedor, unindo a teoria acadêmica com a prática exigida pelo ecossistema de software.

---

### 🛠️ Foco de Aprendizado
Estou mergulhado nos fundamentos que compõem um software de qualidade, priorizando a lógica e a clareza no desenvolvimento.

*   **Fundamentos de Engenharia:** Dedicação ao estudo de Algoritmos, Lógica de Programação e Estruturas de Dados.
*   **Desenvolvimento Web:** Explorando as tecnologias essenciais para a criação de aplicações modernas e funcionais.
*   **Qualidade de Código:** Interessado em padrões de desenvolvimento que facilitem a leitura e a manutenção do software (Clean Code).
*   **Metodologias Ágeis:** Estudando como equipes de tecnologia se organizam para entregar valor de forma eficiente.

### 📚 Conhecimentos em Evolução
*   **Tecnologias:** Lógica de Programação e introdução ao Desenvolvimento Web (HTML/CSS/JS).
*   **Soft Skills:** Comunicação assertiva, proatividade e facilidade em trabalhar em grupo.
*   **Idiomas:** Inglês Básico | Português Nativo.

### 🎯 Objetivo
Busco minha primeira oportunidade como **estagiário ou desenvolvedor trainee**, onde eu possa contribuir com minha energia, aprender com profissionais experientes e aplicar meu conhecimento técnico em projetos reais.
